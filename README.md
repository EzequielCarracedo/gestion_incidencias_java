# Incident Management - Java

Aplicación desarrollada en Java para la gestión básica de usuarios e incidencias,
simulando un entorno de soporte técnico.

## Funcionalidades
- Alta y listado de usuarios
- Creación y gestión de incidencias
- Cambio de estado de incidencias
- Menú interactivo por consola

## Tecnologías
- Java
- Programación Orientada a Objetos
- Git / GitHub

## Objetivo
Proyecto personal realizado como estudiante de Desarrollo de Aplicaciones Multiplataforma (DAM)
para reforzar conocimientos de Java y lógica de programación.
